== Description ==

1. This plugin for showing Out Of Stock Notice on product loop page and shop page.
2. It is having very easy function feel free to edit the functions and css. Happy Coding.
3. You can change the text and colors using admin options Dashboard->No Stock Notice. 


== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress.
2. Extract the zipped file and upload the folder `download-plugin` to `/wp-content/plugins/` directory.


== Frequently Asked Questions ==

How can i use this plugin?
	
	Just download and activate the plugin. In the woocommerce listing page and product detail page 'Sold Out' notice will be shown on product image.
